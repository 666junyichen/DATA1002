# DATA1002 - University Project

This repository contains my DATA1002 university project, which focuses on data analysis and statistical computing.

## Project Overview

DATA1002 is an introductory data science course that covers:
- Data collection and cleaning
- Exploratory data analysis
- Statistical analysis
- Data visualization
- Basic machine learning concepts

## Project Structure

```
DATA1002/
│
├── data/               # Raw and processed data files
├── notebooks/          # Jupyter notebooks for analysis
├── src/                # Python source code
├── reports/            # Generated reports and figures
├── requirements.txt    # Python dependencies
└── README.md          # This file
```

## Getting Started

### Prerequisites

- Python 3.8 or higher
- pip package manager

### Installation

1. Clone this repository:
```bash
git clone https://github.com/666junyichen/DATA1002.git
cd DATA1002
```

2. Install required packages:
```bash
pip install -r requirements.txt
```

### Usage

Open and run the Jupyter notebooks in the `notebooks/` directory:

```bash
jupyter notebook
```

Navigate to the `notebooks/` folder and open any `.ipynb` file to view the analysis.

## Project Components

### Data Analysis
- **Data Exploration**: Initial exploration and understanding of the dataset
- **Data Cleaning**: Handling missing values, outliers, and data quality issues
- **Statistical Analysis**: Descriptive statistics and hypothesis testing
- **Visualization**: Creating informative plots and charts
